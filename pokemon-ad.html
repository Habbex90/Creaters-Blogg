Pokémon Arctic Dark

It's a pokémon fan made game set in Sweden with new fakemon based on Swedish mythology. It has one of the best stories out there if you ask me.

Instruktioner

Function: Keys: Description:
Movement Arrow keys Move up, down, left, right
Confirm / Talk / Use menu selection Z / Enter / Space Select, talk to NPC, interact with objects
Cancel / Back / Close menu X / Esc Go back, close menu
Run Shift Hold to run (if you have Running Shoes)
Open menu X / Esc Opens main menu
Bike (if you have one) A Use bike
Hotkey for selected items D, F, S, A Can be configured to different shortcuts
Save Game In the menu There is no hotkey, save via menu
