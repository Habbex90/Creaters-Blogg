Quadball Mania


This is a sports manager game. You play as a manager of a quadball team. You can create your own or choose an existing one.
There are different tournaments but when you play as a manager of a team (if you are lucky) you only play 3 tournaments the basic tournament (where you start playing),
the Grand Master League (where the 20 best teams play) and the World Cup (world championship).
You have to think about the economy and which players to choose like a normal sports manager game.

Instruktioner

Function: Keys: Description:
Movement Arrow keys Move up, down, left, right
Confirm / Talk / Use menu selection Z / Enter / Space Select, talk to NPC, interact with objects
Cancel / Back / Close menu X / Esc Go back, close menu
Run Shift Hold to run (if you have Running Shoes)
Open menu X / Esc Opens main menu
Bike (if you have one) A Use bike
Hotkey for selected items D, F, S, A Can be configured to different shortcuts
Save Game In the menu There is no hotkey, save via menu
